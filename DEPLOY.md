# Deploy Agriमूल्य (Vercel + Render + Neon + Upstash)

## Architecture

| Service | Host | Free tier |
|---------|------|-----------|
| Frontend | [Vercel](https://vercel.com) | Yes |
| Backend API | [Render](https://render.com) | Yes (sleeps when idle) |
| PostgreSQL | [Neon](https://neon.tech) | Yes |
| Redis | [Upstash](https://upstash.com) | Yes |
| Price poll cron | [cron-job.org](https://cron-job.org) | Yes |

---

## 1. Database (Neon)

1. Create a Neon project and copy `DATABASE_URL`.
2. Run migrations:

```bash
psql "$DATABASE_URL" -f backend/src/db/schema.sql
psql "$DATABASE_URL" -f backend/src/db/migrations/001_prices_unique.sql
```

---

## 2. Redis (Upstash)

1. Create a Redis database on Upstash.
2. Copy the **Redis URL** (`rediss://...`).

---

## 3. Backend (Render)

1. Push this repo to GitHub.
2. Render → **New → Blueprint** → select repo (`render.yaml`).
3. Set these secrets when prompted:

| Variable | Example |
|----------|---------|
| `DATABASE_URL` | Neon connection string |
| `REDIS_URL` | Upstash URL |
| `CORS_ORIGIN` | `https://your-app.vercel.app` |
| `AGMARKNET_API_KEY` | data.gov.in key |

`JWT_SECRET` and `CRON_SECRET` are auto-generated by Render.

4. Note your API URL: `https://agritech-api.onrender.com`

`DISABLE_CRON=true` on Render — use external cron instead (step 5).

---

## 4. Frontend (Vercel)

1. [vercel.com](https://vercel.com) → **Add New Project** → import GitHub repo.
2. Settings:

| Setting | Value |
|---------|--------|
| Root Directory | `frontend` |
| Framework | Vite |

3. Environment variable (Production):

```
VITE_API_URL=https://agritech-api.onrender.com/api
```

4. Deploy. URL example: `https://agritech.vercel.app`

5. Update Render `CORS_ORIGIN` to match your Vercel URL, then redeploy backend.

Preview deployments: set `CORS_ORIGIN` to comma-separated URLs:

```
https://agritech.vercel.app,https://agritech-git-main-you.vercel.app
```

---

## 5. External cron (cron-job.org)

Keeps Render awake and triggers price polling every 15 minutes.

1. Create account at [cron-job.org](https://cron-job.org).
2. New cron job — every **14 minutes**:
   - URL: `https://agritech-api.onrender.com/api/admin/trigger-poll?secret=YOUR_CRON_SECRET`
   - Method: GET
3. Copy `CRON_SECRET` from Render env vars.

Optional wake-up ping (every 10 min):

```
GET https://agritech-api.onrender.com/api/health
```

---

## 6. Smoke test

```bash
curl https://agritech-api.onrender.com/api/health
curl "https://agritech-api.onrender.com/api/prices?commodity=Wheat&limit=3"
```

Open Vercel URL → prices load → register → login → watchlist.
